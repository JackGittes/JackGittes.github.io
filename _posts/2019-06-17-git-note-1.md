|          Git命令         |   选项   |            作用            |
|:------------------------:|:--------:|:--------------------------:|
| add [file or dir]        | 空       | 将文件加入追踪             |
| commit                   | 空       | 提交                       |
|                          | -m       | 提交同时添加说明信息       |
| rm [file]                | 空       | 从索引与文件系统中移除文件 |
|                          | --cached | 仅从索引中移除文件         |
| mv [old file] [new file] | 空       | 重命名                     |
| log                      | 空       | 显示版本库日志信息         |
|                          | -n       | 输出限制为n个提交          |
|                          | --stat   | 显示提交统计信息           |
| show                     | 空       | 查看某个提交               |
| show-branch              | 空       | 显示分支                   |
|                          | -a       | 显示全部分支               |
|                          | -r       | 显示远程分支               |
| branch                   |          |                            |
| merge-base               |          |                            |
|                          |          |                            |
|                          |          |                            |
